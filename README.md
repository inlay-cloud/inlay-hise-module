# Inlay HISE Module

Inlay HISE Module is a lightweight HISE Script integration for protecting a plugin with Inlay activation and local access validation. It provides a runtime unlocker that manages browser-based activation, offline access-token validation and refresh, plugin lock-state broadcasts, protected expansion entitlements, and optional default UI for blocking access until the product is unlocked.

Module overview: [`docs/module-overview.md`](docs/module-overview.md).
Expansions protection guide: [`docs/expansions-protection-guide.md`](docs/expansions-protection-guide.md).

## Repository structure

- `docs/` — Module documentation and usage guides.
- `examples/` — Example HISE projects demonstrating module integration.
- `scripts/` — Shell scripts for installation, release preparation, and publishing.
- `src/` — HISE Script source files that implement the module.
- `tests/` — HISE test project fixtures and test-related assets.
